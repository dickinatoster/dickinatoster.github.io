聊天應用 — ntfy.sh 即時通知整合

技術：HTML / CSS / JavaScript / ntfy.sh / PowerShell

這個範例示範如何使用 ntfy.sh 提供簡單的 Publish/Subscribe 通知功能，並把它嵌入單頁聊天介面。使用者可以直接於瀏覽器連線訂閱主題，或由腳本與 VS Code 任務發送通知。

特色

  • HTTPS 發送（POST）到 ntfy.sh
  • Server-Sent Events (SSE) 訂閱（/sse）即時接收
  • 支援中文主題（URL 編碼）與錯誤處理
  • 含 PowerShell 腳本 `scripts/send-ntfy.ps1` 與 VS Code 任務範本

快速示範

在本機開啟此專案後，按下連結即可在新分頁開啟聊天應用：

打開聊天應用（5/index.html）
5. ntfy 聊天應用
單頁聊天 UI，整合 ntfy.sh 發送與 SSE 訂閱

命令列發送示例

pwsh -NoProfile -ExecutionPolicy Bypass -File .\scripts\send-ntfy.ps1 -Server "https://ntfy.sh" -Topic "drher" -Title "怡文" -Message "測試通知"

注意

若使用自有 ntfy 伺服器，請在 Server 欄輸入完整 URL（例如 https://ntfy.example.com）。若使用本機的自訂主機名稱（如 drher），請確保 DNS/hosts 有對應。

檔案

  • 5/index.html — 聊天應用主頁
  • scripts/send-ntfy.ps1 — PowerShell 發送腳本
  • .vscode/tasks.json — VS Code 任務設定
